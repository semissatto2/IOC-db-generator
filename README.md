# IOC-db-generator
___
	Código para geração automática de arquivo de configuração de variáveis EPICS para s7plc. 
	O código também imprime (no terminal) o tamanho total do DataBlock
	
	
	Execução:
	$ db-generator.py <arquivoEntrada.csv> <example.db>
